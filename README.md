# MiDraw란?

   * MIDI와 DRAW의 합성어로 점을 찍어서 그림을 그리듯이 악보를 작성하여 연주 시키는 프로그램 입니다.

# MiDraw 사용법

   * 피아노, 기타, 베이스, 드럼을 선택 후 악보에 마음대로 그리면 됩니다.
   * 조금 더 전문적으로 그릴수 있도록 계이름과 음, 박자 선을 제공합니다.
   * 저장 파일은 독자적 파일 포멧인 .esc 파일 이며, 웹을 통해 자신이 만든 악보를 공유할수 있습니다.
